# 💰 Finance Dashboard UI

A simple and clean Finance Dashboard built using React.
This project focuses on displaying financial data in an easy-to-understand and interactive way using basic frontend concepts.

## 🚀 Live Demo

https://zorvyn-finance-dashboard-ui-six.vercel.app

## 📌 Features

* Dashboard overview with:

  * Total Balance
  * Total Income
  * Total Expenses

* 📊 Simple Charts:

  * Monthly trend visualization
  * Category-based spending breakdown

* 📋 Transactions Section:

  * Shows date, amount, category, and type
  * Search and filter functionality

* 👤 Role-Based UI:

  * Viewer → can only view data
  * Admin → can see "Add Transaction" button

* 💡 Insights Section:

  * Highest spending category
  * Total income and expenses

* 📱 Responsive design for different screen sizes

## 🛠️ Tech Stack

* React (with Vite)
* JavaScript (ES6)
* HTML & CSS

## 🧠 Approach

I built this project using basic React concepts like functional components, useState, props, and array methods (map and filter).
For charts, I avoided using external libraries and created simple visualizations using div elements and dynamic styling. This keeps the project lightweight and easy to understand.
The focus was on keeping the UI clean, readable, and beginner-friendly while still covering all required features.

## 📌 Notes

* This project uses mock data (no backend)

* Can be extended with API integration or advanced charts
